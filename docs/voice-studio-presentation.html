05 / 09 — 详解

声音复刻
怎么用？

依赖 QWEN 提供的 CosyVoice 服务， 支持录音或上传现成音频两种方式。

⚠️ 关键点：克隆 API 要求声音必须有可访问外链， 所以工具需要后端配置 OBS + 域名。
  • A 有 OBS / 域名：直接在工具里录音或上传，自动走完外链流程。
  • B 没有后端存储：把原始音频传到 tmpfile.link 等免认证临时存储。
  • C 把得到的文件链接填回工具，触发 CosyVoice 克隆。
  • D 克隆完成后，得到一个属于你的专属 voice ID，下一步合成会用到。

合成路径

把写好的稿子变成配音，付费和免费各走一条。

💡 关于 SSML：CosyVoice 支持 SSML 标记， 但目前需要手写。后续会加上可视化编辑，方便生成有趣对话。
方式 01 · 用克隆声音
CosyVoice 合成
用上一步复刻的声音生成配音。新账号有免费 token，付费使用也很便宜。
方式 02 · 零成本
Edge-TTS FREE
完全免费、不需要克隆声音。抖音上刷到的不少配音，背后就是 Edge-TTS。

09 / 09 — 详解

字幕生成
怎么跑？

使用本地的 fast-whisper 模型进行语音识别， 直接保存为标准 .srt 字幕文件。

🎯 亮点：支持把上一步合成的多段语音拼装识别， 一次性得到完整的时间轴。
  • A 直接上传任意要识别的语音/视频文件，一键转字幕。
  • B 或者选择刚刚合成的语音，工具会自动拼装多段音频。
  • C 输出标准 .srt，直接导入 Camtasia / Premiere / 剪映。
